Concept ⟷ Example

Left · Concept

One theme = a set of CSS variables. Colors, fonts, radius, shadows all go in :root.

  • — Tokens define semantics, no hardcoded colors
  • — base.css uses tokens for layout
  • — Each theme only changes variables

Right · Example

:root {
  --bg: #fff;
  --text-1: #0c0d10;
  --accent: #3b6cff;
  --radius: 18px;
}
.card {
  background: var(--bg);
  color: var(--text-1);
  border-radius: var(--radius);
}
