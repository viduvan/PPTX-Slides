Trade-offs · Honest trade-offs

Pure HTML presentations：What's good，What's painful

✓ Pros

  • Zero build: runs from a single file
  • Diffable: Git-friendly, easy to review
  • Programmable: fully customizable animations
  • Distributable: URL / PDF / PNG all work

✗ Pain points

  • Collaboration is not real-time like Keynote
  • Steeper learning curve for non-developers
  • Complex animations still require writing JS
  • Some fonts missing in offline environments
